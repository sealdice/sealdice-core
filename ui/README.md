[SealDice 海豹核心](https://github.com/sealdice/sealdice-core) 的前端实现。

## 说明

本仓库为海豹核心的前端实现，使用 Vue3 + ElementPlus + tailwindcss 开发。

## 部署

```bash
pnpm install
```

### 开发环境

```bash
pnpm run dev
```

### 生产环境

```bash
pnpm run build
```
